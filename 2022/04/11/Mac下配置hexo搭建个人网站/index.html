Mac下配置hexo搭建个人网站

一、环境准备

1. 安装node.js

1.1 官网下载安装：https://nodejs.org/en/

1.2 双击刚下载的文件，按照默认路径安装

1.3 安装完成后打开终端，输入

node -v
npm -v
出现版本号即代表安装成功

二、开始安装Hexo

1. 安装hexo

npm install -g hexo-cli

如果npm安装缓慢可以用cnpm安装

安装完成可输入hexo -v查看版本
